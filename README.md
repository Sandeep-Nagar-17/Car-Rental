# Car-Rental

##  Features

- **User Authentication** — Secure sign-up and login using JWT.
- **Car Listings** — View a variety of available rentals with details including model, price, and availability.
- **Search & Filters** — Quickly locate cars by type, model, or price range.
- **Booking System** — Select dates, calculate pricing, and complete reservations.
- **Responsive Design** — Optimized for both desktop and mobile devices.
- **Image Optimization** — Fast, high-quality images powered by ImageKit CDN.
- **Admin Panel** — Manage cars, users, and bookings
